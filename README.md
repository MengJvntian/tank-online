# 坦克小英雄 Plus 🚀

> 一个支持单机、多人联机、手机/电脑双端适配的网页坦克大战小游戏。

# 🎮 游戏简介

《坦克小英雄 Plus》是一款基于：

- HTML5 Canvas
- JavaScript
- Node.js
- WebSocket

开发的轻量级在线坦克大战小游戏。

游戏支持：

- 单机闯关
- 多人联机
- 手机/电脑双端适配
- 房间系统
- 职业技能
- Boss 战
- 商店成长系统
- 本地存档

玩家既可以单机刷金币挑战高难关卡，也可以邀请朋友一起联机守卫基地。

# 🕹️ 游戏特色

## 单机模式

- 多关卡挑战
- 解锁新地图
- 获取金币
- 解锁皮肤与升级

- 最多 4 人联机
- 创建公开/私密房间
